# PWA Fase 1 (Installable + Ícone) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `analytis.zyntra.company` installable as a PWA from iOS Safari, with the project's gradient "A" icon and standalone (no-Safari-chrome) launch behavior.

**Architecture:** Pure static-asset addition — no service worker, no runtime code. We ship 6 PNG icons (1024 source + 5 generated variants), a `manifest.webmanifest`, and Apple-specific `<meta>` tags in `index.html`. Icon generation runs once locally via a Node script using `sharp`; outputs are committed. The Vite build pipeline is untouched.

**Tech Stack:**
- `sharp` (added as devDependency, ~30 MB, not in production bundle) — image resize
- Static JSON manifest served from `frontend/public/`
- HTML meta tags read by Safari iOS ≥ 16.4

**Spec:** `docs/superpowers/specs/2026-06-20-pwa-fase1-design.md`

### Task 3: Write the icon generation script

**Files:**
- Create: `frontend/scripts/generate-icons.mjs`

- [ ] **Step 1: Create the script**

Write this file (Vite serves anything under `public/` at the URL root, so we write directly to `frontend/public/`):

```javascript
// frontend/scripts/generate-icons.mjs
// One-shot resize: reads public/icon-1024.png and writes 5 derived PNGs.
// Run: pnpm gen:icons

import { promises as fs } from "node:fs";
import path from "node:path";
import { fileURLToPath } from "node:url";
import sharp from "sharp";

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const publicDir = path.resolve(__dirname, "..", "public");
const source = path.join(publicDir, "icon-1024.png");

const SIZES = [180, 192, 256, 384, 512];

async function main() {
  try {
    await fs.access(source);
  } catch {
    console.error(
      `[gen:icons] source not found: ${source}\n` +
      `  Copy your 1024x1024 PNG to public/icon-1024.png and re-run.`
    );
    process.exit(1);
  }

  const meta = await sharp(source).metadata();
  if (meta.width !== meta.height) {
    console.error(
      `[gen:icons] source must be square, got ${meta.width}x${meta.height}`
    );
    process.exit(1);
  }
  if (meta.width < 512) {
    console.error(
      `[gen:icons] source must be at least 512x512, got ${meta.width}x${meta.height}`
    );
    process.exit(1);
  }

  await Promise.all(
    SIZES.map(async (size) => {
      const out = path.join(publicDir, `icon-${size}.png`);
      await sharp(source).resize(size, size, { fit: "cover" }).png().toFile(out);
      console.log(`[gen:icons] wrote ${out}`);
    })
  );

  console.log(`[gen:icons] done — ${SIZES.length} files written`);
}

main().catch((err) => {
  console.error("[gen:icons] failed:", err);
  process.exit(1);
});
```
